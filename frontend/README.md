# WeatherAPI Frontend

React + Vite frontend for the WeatherAPI weather dashboard.

## Features

- City search input
- Weather card UI
- Temperature unit toggle
- Weather icon display
- Responsive layout

## Setup

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

## Environment Variables

```env
VITE_API_BASE_URL=http://localhost:8000
```
